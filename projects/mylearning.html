MyLearning

Elementor • HTML • CSS • JS

Overview

MyLearning is a WordPress application that gives a platform for the employees at Otto International (Oi) to learn and broaden their knowledge about digitalisation, learning and development, and the company's values. It was developed as an individual project that slowly expanded to the addition of a designer to supply graphical content.

Components

Front-End

The first version of MyLearning was inspired by Microsoft's support homepage design. Using CSS, I have partitioned the sections to a hero, categories, and featured content. Since MyLearning is built on WordPress, one of the many posts functions was used to automatically to update the front page with the latest training posts. This way, employees can easily find the newly released training materials and stay up to date.

For the second version of MyLearning, Elementor was adopted to handle the frontend. It brought a more creative touch to the project. Since Elementor is tightly integrated with WordPress, it has made frontend development much faster and smoother.

Learning Management System (LMS)

E-learning was the fundamental pillar in creating MyLearning. A place for Oi users to practise learning and development by content curated from the training team. LearnDash proved to be most useful to the given requirements that consists of courses,lessons, topics, quizzes, and certificates. With its comprehensive documentation, it has helped in creating aninternal app using their REST API. While working on the mobile port of MyLearning with an App Developer, I have created mock ups and a visual guide on how the API endpoints are used to speed up the development process.
